# Claude Development Notes for Charades App

## Package Manager
- **Use bun instead of npm** for all package management operations
- This project uses bun for faster installs and better performance
- Commands: `bun install`, `bun run dev`, `bun run build`, etc.

## Project Structure
- This is a Capacitor app for iOS and Android
- Uses React + TypeScript + Vite
- Prefer Capacitor APIs over web APIs for mobile functionality
